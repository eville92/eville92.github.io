# Functions

## Description
The lisp program computes a 3D distance from origin. The user inputs an `x` parameter, a `y` parameter, and a `z` parameter. 
The program then squares the variables `x = x * x`, `y = y * y`, and `z = z * z`, adds up the new variable products `x + y + z`, square 
roots the sum `sqrt(x + y + z)` and prints the result. 

## Notes
Can compute 2D distances as the `z` variable is optional.
