# Comment Reader

## Description
A program that reads comments. When executed, the program will open `comment.txt` by default. 

## Related Works
This program uses my [Heapsort](https://github.com/eville92/eville92.github.io/tree/main/projects/heapsort) implementation, which fills an array with 100 random numbers and prints the array before and after max‑heapifying (see `comment1.txt` test case).

## Notes
- Limited to reading block comments `/* text within here */`. 
- Test cases folder provided.
