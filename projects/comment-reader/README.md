# Comment Reader

## Description
A program that reads comments. When executed, the program will open `comment.txt` by default. 

## Related Works
This program uses my [Heapsort](https://github.com/eville92/eville92.github.io/tree/main/projects/heapsort) script written in a .txt file (see `comment1.txt` in `test-case`).

## Notes
- Limited to reading block comments `/* text within here */`. 
- Test cases folder provided. 
- It's suggested to copy scripts and paste them in `comment.txt`.
