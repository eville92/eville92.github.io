# Games

## Description
My first progamming project created in 2023! 
This python program consists of a small library of two games: 
- Paper, Rock, Scissors! 
- Battle Cards 

When the player wins against the computer/ai, the program generates a high-score log:
- `PRS high score.txt` for Paper, Rock, Scissors!
- `BC high score.txt` for Battle Cards

The logs are appended every time the player wins.
